# folie zabezpieczające

Cienkie warstwy ochronne naklejane na powierzchnie, np. meble, szyby, ekrany lub elementy lakierowane. Przy czyszczeniu wymagają delikatnych preparatów, które nie odkleją folii, nie zmatowią jej i nie uszkodzą kleju.
