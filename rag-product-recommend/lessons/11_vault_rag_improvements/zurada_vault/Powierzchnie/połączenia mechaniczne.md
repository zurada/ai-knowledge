# połączenia mechaniczne

Miejsca łączenia części maszyn, takie jak zawiasy, mocowania, śruby czy zaciski. Wymagają ostrożnego czyszczenia, aby nie uszkodzić elementów roboczych i nie pogorszyć ich działania.
