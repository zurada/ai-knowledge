# tapicerka samochodowa

Obicia foteli i elementów wnętrza pojazdu wykonane z tkanin, weluru, mikrofibry lub mieszanek materiałów. Wymaga czyszczenia o kontrolowanej wilgotności, aby nie przemoczyć pianek, nie zostawić zacieków i nie uszkodzić elementów elektrycznych foteli.
