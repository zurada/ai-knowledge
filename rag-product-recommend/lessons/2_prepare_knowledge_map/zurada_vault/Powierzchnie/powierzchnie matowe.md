# powierzchnie matowe

Powierzchnie bez połysku, np. matowe płytki, laminaty lub lakierowane elementy, na których łatwo widać smugi i wybłyszczenia. Wymagają łagodnych środków bez nadmiernego nabłyszczania i bez agresywnego szorowania.
