# powierzchnie matowe

Powierzchnie o niepołyskliwym wykończeniu, np. meble, płytki, laminaty lub powłoki lakierowane. Wymagają środków, które nie nabłyszczają punktowo, nie zostawiają smug i nie wycierają struktury wykończenia.
